### ASCII Art

For example, the following defines a simple AsciiArt diagram:

````
```svgbob {type=svgbob format=svg}
  _____
 /  O__]
/  ___]
\   \
 |   |
 |   |
 /   /
 \   \
 /   /
```
````

```svgbob {type=svgbob format=svg disabled=false}
  _____
 /  O__]
/  ___]
\   \
 |   |
 |   |
 /   /
 \   \
 /   /
```
